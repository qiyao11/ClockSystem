# **子母钟管理系统**

**本项目为控制塔钟子母钟所制作的管理系统，用于安装在塔钟服务器上**

**系统包含（将实现）以下功能**

###### **母钟时间**

母钟基础时间显示（时 / 分 / 秒、日期 / 星期）✅

母钟时间调整✅

###### **子钟控制**

子钟硬件交互❌

子钟微调❌

子钟反馈调节❌

###### **钟表灯光**

定时开关钟表灯光✅

###### **报时设置**

自定义报时✅

报时广告❌

###### **核心设计原则**

优先保障母钟时间基准的准确性，子钟控制功能基于母钟基准实现分布式同步校准；

模块化设计，便于后续功能拓展与维护；

配置化管理，核心功能支持自定义规则配置，适配不同场景需求。

###### **后续规划**

完成子钟控制全量功能开发与硬件联调；

新增子钟故障告警模块（支持短信 / 邮件 / 系统内告警）；

增加数据统计分析模块，统计灯光运行时长、报时次数、子钟同步偏差率等数据；

优化前端交互界面，提升操作便捷性；

完善系统日志与异常监控机制。

###### **部署说明**

该系统部署于塔钟服务器端，适配Windows服务器系统，依赖环境与部署文档将随功能迭代逐步完善。
