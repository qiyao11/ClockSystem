# 贡献指南

欢迎为子母钟管理系统项目贡献代码和建议！以下是参与项目的指南。

## 开发环境设置

1. 克隆仓库：
   ```bash
   git clone https://github.com/qiyao11/ClockSystem.git
   cd ClockSystem
   ```

2. 安装依赖：
   - Visual Studio 2022 或更高版本
   - .NET 10.0 或更高版本

3. 打开解决方案：
   - 使用 Visual Studio 打开 `ClockSystem.sln` 文件

## 贡献流程

1. 创建新分支：
   ```bash
   git checkout -b feature/your-feature-name
   ```

2. 进行代码修改

3. 提交更改：
   ```bash
   git add .
   git commit -m "Add: 描述你的更改"
   ```

4. 推送分支：
   ```bash
   git push origin feature/your-feature-name
   ```

5. 创建 Pull Request

## 代码规范

- 遵循 C# 编码规范
- 确保代码可读性和可维护性
- 为新功能添加适当的注释
- 确保代码通过所有测试

## 问题报告

如果发现问题，请在 GitHub Issues 中报告，包括：
- 问题描述
- 复现步骤
- 预期行为
- 实际行为
- 环境信息

## 联系我们

如有任何问题或建议，请通过 GitHub Issues 与我们联系。
